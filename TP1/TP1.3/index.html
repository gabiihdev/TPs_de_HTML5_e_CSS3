Como funciona a Web e o que é uma página Web?

A Web é uma rede de documentos hipermídia interligados e executados na internet, a página web é um desses documentos.

Quais são os papéis das linguagens HTML e CSS em uma página Web?

O HTML é uma linguagem de marcação utilizada para inserir o conteúdo de uma página web. O CSS, por outro lado, é utilizado para dar a cara da página, ou seja, é responsável pelo estilo e formatação.

O que são ambientes de programação?

São ambientes para criação e testes de códigos.
